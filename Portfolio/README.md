# Portfolio de Bekkar Khelaifa

Version repensée du portfolio, construite en HTML, CSS et JavaScript.

## Contenu
- `index.html` : page principale du portfolio
- `styles.css` : design moderne responsive
- `script.js` : thème clair/sombre + animations d'apparition
- `assets/docs/` : CV PDF
- `assets/images/` : visuels de projets

## Mise en ligne sur GitHub Pages
1. Remplacer le contenu de l'ancien dépôt par ces fichiers.
2. Commit puis push sur la branche principale.
3. Activer GitHub Pages dans les paramètres du dépôt si besoin.
